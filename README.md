# Veaks - Premium Men's Footwear Marketplace

A modern, luxury landing page for Veaks, a premium men's shoe marketplace built with React, Vite, and Framer Motion.

## Features

- **Responsive Design**: Mobile-first, fully responsive layout for all screen sizes
- **Smooth Animations**: Powered by Framer Motion and AOS (Animate on Scroll)
- **Product Categories**: Sneakers, loafers, dress shoes, boots, sandals, and drivers
- **Featured Products**: Showcase of top picks with pricing
- **Interactive Components**: Hover effects, animations, and Swiper carousels
- **Luxury Aesthetic**: Clean, modern design with premium typography and spacing

## Tech Stack

- **React 19**: UI library
- **Vite**: Build tool and dev server
- **Framer Motion**: Animation library
- **AOS**: Scroll animations
- **Swiper.js**: Carousels and sliders
- **React Icons**: Icon library
- **CSS Modules**: Scoped styling

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

To create a production build:

```bash
npm run build
```

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```
